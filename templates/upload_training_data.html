📋 Panduan Upload Data Training

✅ Format File yang Didukung:

  • ZIP File: Dataset lengkap dengan struktur YOLO
  • Gambar: JPG, JPEG, PNG, BMP, TIFF

📏 Batasan Upload:

  • Ukuran maksimum: 500 MB
  • Resolusi gambar: 100x100 - 4000x4000 pixel
  • Minimal 10 gambar per dataset
  • Format label: YOLO (.txt)

📁 Struktur Dataset YOLO (untuk file ZIP):

dataset.zip
├── images/           # Folder berisi gambar
│   ├── image1.jpg
│   ├── image2.jpg
│   └── ...
└── labels/           # Folder berisi file label
    ├── image1.txt    # Label untuk image1.jpg
    ├── image2.txt    # Label untuk image2.jpg
    └── ...

Format file label (.txt):
class_id x_center y_center width height
Contoh: 0 0.5 0.5 0.3 0.4
(nilai koordinat dalam range 0.0 - 1.0)

💡 Tips untuk Dataset Berkualitas:

  • • Gunakan gambar dengan pencahayaan yang bervariasi
  • • Sertakan berbagai sudut pengambilan foto wajah
  • • Pastikan wajah terlihat jelas dan tidak blur
  • • Semakin banyak data training, semakin akurat model
  • • Untuk hasil terbaik, gunakan minimal 50-100 gambar per orang
